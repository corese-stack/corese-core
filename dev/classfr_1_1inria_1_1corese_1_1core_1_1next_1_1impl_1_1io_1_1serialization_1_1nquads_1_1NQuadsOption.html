Detailed Description

Configuration for N-Quads serialization format.

This class extends AbstractNFamilyOption and provides specific defaults and options tailored for N-Quads, which extends N-Quads with named graphs.

Use the Builder class to create instances of NQuadsConfig. A predefined default configuration is available via defaultConfig().
