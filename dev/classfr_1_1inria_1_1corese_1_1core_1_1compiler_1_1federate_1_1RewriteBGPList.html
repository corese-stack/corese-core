Detailed Description

Rewrite BGP of triple with several URI input:
map: uri -> bgpList create: map: bgp -> uriList where bgp are connected.

when BGP_LIST = true generate list or partition of connected bgp with |bgp|>1 generate service S {bgp} and complete each partition with missing triples

when BGP_LIST = false for each |bgp| > 1 create: exp = service (uriList) {bgp} complete exp with every triple not in bgp: 1) with several uri 2) with one uri return union of exp above

if there is no |bgp|>1 generate service (S) {triple} for every triple
