Detailed Description

An abstract base class for serialization configurations of N-Family RDF formats (e.g., N-Triples, N-Quads).

This class extends AbstractSerializerOption and provides a common foundation for formats that typically have simpler, line-based structures and specific default behaviors regarding literal datatypes and character escaping.

It enforces the use of the Builder pattern for construction through its nested AbstractNFamilyBuilder. Subclasses are expected to extend this configuration and its builder to add format-specific options.
