Detailed Description

DataManager on top of json (or xml) document.

a) implemented in java as graph data manager path = path of insert where query that creates a graph (from json, xml, etc.) update query creates graph data manager

b) implemented in ldscript when path contains ldscript (draft for testing) path = path of ldscript function definition 1- read (json) file with us:read() ldscript determine json file 2- iterate edges with ldscript function us:iterate()
