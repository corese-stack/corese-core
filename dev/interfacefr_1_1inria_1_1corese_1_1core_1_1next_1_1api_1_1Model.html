Detailed Description

This class represents an RDF model, a set of statements.

This is the central class to handle RDF data. Statements can have zero or several resources as contexts to represent the different graphs in the model. A model also handles a set of namespaces. There can be different prefixes for the same namespace, although it is ill-advised, how it affects serialization is implementation-dependant. There cannot be two namespaces with the same prefix. Statements can have zero or several resources as contexts, representing different named graphs within the modela
