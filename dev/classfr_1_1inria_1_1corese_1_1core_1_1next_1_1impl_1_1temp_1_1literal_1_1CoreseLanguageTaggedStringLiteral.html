Detailed Description

An implementation of a language-tagged string literal used by Corese.

This class represents a string literal that is associated with a language tag, specifically when the datatype IRI is http://www.w3.org/1999/02/22-rdf-syntax-ns#langString. It extends AbstractStringLiteral and implements CoreseDatatypeAdapter.
