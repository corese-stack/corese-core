Detailed Description

An abstract base class for all RDF serialization configurations.

This class defines common configuration parameters that are applicable across various RDF serialization formats (e.g., N-Triples, Turtle, RDF/XML).

It enforces the use of the Builder pattern for construction through its nested AbstractBuilder. Subclasses are expected to extend this configuration and its builder to add format-specific options.
