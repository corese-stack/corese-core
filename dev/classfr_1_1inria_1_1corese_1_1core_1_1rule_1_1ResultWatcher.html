Detailed Description

Watch kgram query solutions of rules for RuleEngine 1 Check that a solution at loop n contains at least one edge deduced at loop n-1 (that is a new edge) 2 Eliminate duplicate solution by a select distinct * on construct variables 3 Create Edge directly (without Mapping created) Does not optimize with: exists {} Property Path subquery.
