Detailed Description

Configuration for RDF/XML serialization format.

This class extends AbstractSerializerOption directly as RDF/XML has distinct serialization characteristics not shared by the Turtle or N-Family formats.

Use the Builder class to create instances of XmlConfig. A predefined default configuration is available via defaultConfig().
