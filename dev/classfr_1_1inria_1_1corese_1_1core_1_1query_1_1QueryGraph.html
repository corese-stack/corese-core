Detailed Description

Translate a Graph into a Query Use case: let g = eval(construct {} where {}) let q = query(g) eval(q)

The target Query may itself be a construct

There may be a Visitor that rewrites the Graph on the Fly, e.g. : - bnode to variable - constant to variable - generalize type - eliminate edge
