Detailed Description

An abstract base class for serialization configurations of Turtle Trig RDF formats (e.g., Turtle, TriG).

This class extends AbstractSerializerOption and introduces parameters specific to formats that utilize syntax sugar, pretty-printing, and collection syntax.

It enforces the use of the Builder pattern for construction through its nested AbstractTFamilyBuilder. Subclasses are expected to extend this configuration and its builder to add format-specific options.
