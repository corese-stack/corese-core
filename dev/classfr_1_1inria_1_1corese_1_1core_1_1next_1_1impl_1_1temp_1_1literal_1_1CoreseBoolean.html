Detailed Description

An implementation of the xsd:boolean datatype used by Corese.

The xsd:boolean type represents logical boolean values. The valid values for xsd:boolean are true, false, 0, and 1. Values that are capitalized (e.g. TRUE) or abbreviated (e.g. T) are not valid.
