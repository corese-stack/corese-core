Detailed Description

Interface to adapt an external storage system to Corese.

DataManagerRead for select where SPARQL queries.

This interface is not used by the internal Corese graph, specific DataBroker directly process it.

List< Node >  contexts

Returns an Iterable over all Edges in the graph that match the supplied criteria.

If several edges have the same subject, predicate and object then only one is returned.

If the size of the contexts in parameter is not equal to 1, then the value of Graph of Edges returned does not matter. It can be for example http://ns.inria.fr/corese.core.kgram/default. Be careful, the value of Graph of Edges cannot be null.

All null in context list are ignored. E.g. If the list contains only null, it is handled as an empty list.

Parameters
subjectThe subject of the edges to match, null to match edges with any subject.
predicateThe predicate of the edges to match, null to match edges with any predicate.
objectThe object of the edges to match, null to match edges with any object.
contextsThe contexts of the edges to match, null or empty list to match edges with any contexts. If one or more contexts are specified, edges with a context matching any one of these will match.
Returns
An Iterable over all the edges in the graph that match the specified pattern.
