Detailed Description

Configuration for Turtle serialization format.

This class extends AbstractTFamilyOption and provides specific defaults and options tailored for Turtle, such as using collections and anonymous blank nodes.

Use the Builder class to create instances of TurtleConfig. A predefined default configuration is available via defaultConfig().
