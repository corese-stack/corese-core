Detailed Description

An implementation for estimating the selectivity based on statistics data.

This implementation utilize simple multiplication of each variable (bound or unbound) in a triple pattern, namely, sel(triple) = sel(s) * sel(p) * sel(o)

(TODO: A more sophisticated implementation by considering jointed triple pattern should be implemented to imporve the drawback of simple multiplication)
