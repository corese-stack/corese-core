Transitive closure in a rule base This code emulates a transitive closure rule, it is called at each loop in the rule engine When loop == 0 consider all edges When loop > 0, one of the two edges (e1, e2) must have been created at previous loop First step n == 0: all edges Following steps: new edges from previous step loopIndex: index of rule current loop prevIndex: index of rule previous loop created edges are tagged with current loop index.
