Detailed Description

Configuration for N-Triples serialization format.

This class extends AbstractNFamilyOption and provides specific defaults and options tailored for N-Triples, which is a simple, line-oriented format.

Use the Builder class to create instances of NTriplesConfig. A predefined default configuration is available via defaultConfig().
