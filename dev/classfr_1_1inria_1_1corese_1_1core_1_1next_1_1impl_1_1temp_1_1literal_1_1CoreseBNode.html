Detailed Description

An implementation of a blank node (BNode) used by Corese.

A blank node (BNode) represents an unnamed node in an RDF graph, typically used to represent resources that do not have a globally unique identifier (IRI). This class extends AbstractBNode and provides constructors for creating a BNode either from a Corese blank node object or a given string identifier.
