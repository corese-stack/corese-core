Detailed Description

?x rdf:resf * /rdf:first

write paths in a synchronized buffer

?x ^(p/q) ?y ::= ?y p/q ?x -> ?x inv(q)/inv(p) ?y

?x p/q uri path is computed backward from uri to ?x with index=1 other=0 isReverse=true sequence is eval as q/p, all other exp are the same

?x ^(p/q) uri ::= uri p/q ?x -> ?x inv(q)/inv(p) uri

PP Extensions

Path Variable: ?x exp :: $path ?y

Path Length: pathLength($path)

Path Enumeration: ?x exp :: $path ?y graph $path {?a ?p ?b}

Shortest path: ?x distinct short exp ?y ; ?x short exp ?y – TODO: complete short to get only shortest

Path weight: ?x (rdf:first@2 / rdf:rest@1* / ^rdf:first@2) * ?y

Constaint: ?x exp ?this a foaf:Person} ?y ?x exp @[a foaf:Person] ?y

Parallel Path: ?x (foaf:knows || ^rdfs:seeAlso) + ?y

Check Loop: pf.setCheckLoop(true) => exp+ exp{n,m} without loop exec.setPathLoop(false) pragma {kg:path kg:loop false}
