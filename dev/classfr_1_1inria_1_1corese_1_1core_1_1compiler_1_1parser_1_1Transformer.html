Compiler of SPARQL AST to KGRAM Exp Query Use Corese.Core.Sparql parser Use an abstract compiler to generate target edge/node/filter implementations.

sub query compiled as distinct edge/node to avoid inappropriate type inference on nodes, except variable nodes which are shared
