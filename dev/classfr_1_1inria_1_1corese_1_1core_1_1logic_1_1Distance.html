Detailed Description

Semantic distance & similarity with Corese 2.4 Algorithm Extended to property hierarchy distance: kg:pSimilarity.

PRAGMA: need a graph with Node set/get Depth need a root class, default is rdfs:Resource or owl:Thing

If a class is not subClassOf root, depth simulate to 1 Exploit owl:sameAs & owl:equivalentClass/Property

Generalized to property hierarchy with a step of 1/8^n instead of 1/2^n steps can be changed with pragma: kg:similarity kg:cstep 3.0 kg:similarity kg:pstep 4.0

TODO: Expensive with Entailment because of rdf:type generated rdfs:domain and rdfs:range A resource may have several types, in Corese 2.4 types were synthetized into one type
