Detailed Description

Manager for Exception throwned during query processing Manager generates Turtle representation of exceptions.

1) local exceptions managed as a list in Query AST Context 2) remote endpoint service exception managed as list of URI of Turtle document in Query AST Context URI inserted in SPARQL Query Results XML Format link href and stored in Mappings

API: LogManager man = exec.getLogManager(map);
