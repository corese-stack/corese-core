Detailed Description

Configuration for TriG serialization format.

This class extends AbstractTFamilyOption and provides specific defaults and options tailored for TriG, which extends Turtle with named graphs.

Use the Builder class to create instances of TriGConfig. A predefined default configuration is available via defaultConfig().
