inline void imports(String uri)#

Imports an RDF resource from the given URI if it has not been loaded yet.

This method ensures that the URI is not imported multiple times by checking the importedURIs map. If the URI is new, it is added to the map and processed using basicImport(uri).

The current BuildImpl instance is temporarily saved and restored after the import to maintain consistency.

Parameters:

uri The URI of the RDF resource to import.
