class QuerySolverVisitorBasic : public PointerObject, public ProcessVisitor#

Callback manager for LDScript functions with specific annotations Eval SPARQL processor calls before() and after()

@before function us:before(?q) {} @after function us:after(?m) {} @produce function us:produce(?q) {} @candidate function us:candidate(?q, ?e) {}

Author

Olivier Corby, Wimmics INRIA I3S, 2018

Return:

function us:result(?m) {} @solution function us:solution(?m) {}
