class Property#

Corese configuration with properties such as: LOAD_IN_DEFAULT_GRAPH Usage: Property.set(LOAD_IN_DEFAULT_GRAPH, true); Property.load(property-file-path); Property.init(graph); corese-gui.jar -init property-file-path corese-server.jar -init property-file-path A property file example is in core resources/data/corese/property.properties Define variable: VARIABLE = $home=/user/home/ LOAD_DATASET = $home/file.ttl.

Olivier Corby
